鼓励计划（废弃,板子题）

DZ要考六级口语了，但DZ不爱学习，整天游手好闲，无所事事，刚开始妄想裸考过去，没有任何准备。但事实很残酷，大家都知道这次六级会比以前难（虚构）。考试临头了，DZ才明白这个道理（妄想裸考是不可能的），但由于不知道怎么准备，DZ现在十分苦恼。DZ的室友是个热心肠，他想帮助DZ，于是他决定鼓励DZ多学习英语。

室友苦思冥想，想到了如下做法：

  • 室友决定给DZ进行为期n天的鼓励，但在这n天内限制DZ最多只能说m句英语（不能过度学习）。

  • 第 $ i $ 天允许DZ说 aia_iai​(1≤i≤n1 \le i \le n1≤i≤n)句英语 或者不说。

  • 由于DZ说的英语每天都不同，室友会根据当天的情况对DZ进行奖励

    更确切地说，如果DZ在第 iii 天说了英语，每说一句DZ就会得到奖励值 bib_ibi​ (1≤i≤n1 \le i \le n1≤i≤n)。规定DZ初始奖励值为 000 。

DZ为了得到更多奖励会尽可能多说英语，但不会超过m句。室友虽然是热心肠，但他并不确定对DZ的鼓励是否有效，于是他设置了一个奖励值的阈值M,若DZ的奖励值能达到或者超过阈值，则可认为鼓励成功，否则鼓励是失败的。

输入

第一行 输入四个数 n,m,M分别表示：DZ的改造天数n，DZ最多只能说m句英语，以及奖励值阈值M。

第二行 输入n个正整数 aia_iai​，表示第 iii 天允许DZ说的英语数量。

第二行 输入n个正整数 bib_ibi​，表示第 iii 天DZ说一句英语会得到的奖励值。

输出

若鼓励行动失败则输出"Failed"否则输出"Successful",注意：不包含引号""。

数据范围

1≤n≤1031 \le n \le 10^31≤n≤103,1≤m≤1041 \le m \le 10^41≤m≤104,1≤M≤10181 \le M \le 10^{18}1≤M≤1018.

1≤ai≤1051 \le a_i \le 10^51≤ai​≤105，1≤bi≤1051 \le b_i \le 10^{5}1≤bi​≤105.

时间限制

c、c++1sc、c++ 1sc、c++1s.

其他语言 2s2s2s.

样例输出1

  • 样例解释：选择第 1、3、4、5、6天，恰好为 10 句，得到最大奖励值60，但小于阈值100，因此失败了。

样例输入2

样例输出2

  • 样例解释：选择第 2、3天或者第3、4天，DZ说了8句英语，得到最大奖励值34，大于阈值21，因此成功了。

Solution(01背包)：

题目是很明显的01背包，在保证DZ 说的英语不超过mmm 句的情况下， 动态规划求其最大的奖励值，若最大值都小于阈值则鼓励失败。

  • 第一行两个正整数， nnn 和 kkk ：

  • 第二行 nnn个 数，表示给定的序列元素 aia_iai​ 。

1≤n≤2⋅105,0≤k≤61\le n\le2·10^5,0\le k\le61≤n≤2⋅105,0≤k≤6，0≤ai≤63(i=1,2,...,n)0\le a_i \le 63(i=1,2,...,n)0≤ai​≤63(i=1,2,...,n)

    • [2][2][2]：其与和为101010（二进制）,仅包含一个111，不为666，所以对答案贡献为零
    • [2,4][2,4][2,4]：与和为 000 ,同理，贡献为零。
    • [2,1][2,1][2,1]：与和结果000
    • [2,4,1][2,4,1][2,4,1]：与和结果000
    • [4][4][4]：与和结果100100100
    • [4,1][4,1][4,1]：与和结果000
    • [1][1][1]：与和结果111

Solution：

由于序列元素aia_iai​的数据范围很小，且与运算的性质（越与越小），与和结果只能在000 ~ 636363 间，可以考虑枚举000 ~ 636363 的所有可能，然后动态规划求解。

现定义 dp[i][j]dp[i][j]dp[i][j] 表示前 iii 个元素的子序列中与和为 jjj 的个数,考虑当前的元素 aia_iai​ 对哪些情况有贡献即可，则有以下转移关系：

  • 对于当前元素 aia_iai​ ，若选择，则有：
    • dp[i][a[i]&j] = dp[i][j&a[i]] + dp[i-1][j],表示选择当前元素aia_iai​对与和为 a[i]&j的贡献。
  • 若不选 aia_iai​ ：
    • dp[i][j] = dp[i][j] + d[i-1][j]
  • 同时，不管选不选 aia_iai​ ,当 aia_iai​单独为一个序列时，对dp[i][a[i]]也有贡献：
    • dp[i][a[i]] = dp[i][a[i]] + 1

最后统计下前 nnn 个元素的子序列与和在 000 ~ 636363 中，有多少二进制表示下有 kkk 位为1的就行。
